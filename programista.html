Technik Programista

Profil kształcenia ukierunkowany na inżynierię oprogramowania. Program realizowany zgodnie z podstawą programową MEN, rozszerzony o standardy rynkowe.

Standardy Kształcenia

Szczegółowy zakres materiału realizowanego w cyklu 5-letnim.

Języki Programowania

  • Python (Analiza danych, AI, Backend)
  • C# / .NET (Aplikacje desktopowe)
  • C++ (Podstawy algorytmiki, Silniki gier)
  • JavaScript / TypeScript (Frontend)

Technologie Webowe

  • React.js & Node.js
  • HTML5 & CSS3 (Flexbox, Grid)
  • SQL & Relacyjne Bazy Danych (MySQL)
  • Systemy CMS (WordPress)

Metodologia Pracy

  • System kontroli wersji Git & GitHub
  • Zarządzanie projektem (Jira, Trello)
  • Praca w metodyce Agile/Scrum
  • Dokumentacja techniczna

Kwalifikacje Państwowe

Absolwent kończy edukację z tytułem zawodowym Technika Programisty po zdaniu dwóch egzaminów państwowych:

  • INF.03: Tworzenie i administrowanie stronami i aplikacjami internetowymi oraz bazami danych.
  • INF.04: Projektowanie, programowanie i testowanie aplikacji (desktopowych i mobilnych).

Organizacja Zajęć

Bloki Projektowe

Zajęcia zawodowe odbywają się w blokach 3-4 godzinnych, co pozwala na realizację złożonych zadań programistycznych bez przerywania pracy.

Infrastruktura

Każde stanowisko wyposażone jest w stację roboczą klasy enterprise, dwa monitory oraz dostęp do profesjonalnego oprogramowania (JetBrains, Visual Studio).

Kadra Dydaktyczna

Zajęcia prowadzone są przez praktyków z branży IT oraz wykładowców akademickich Politechniki Krakowskiej.

Informacje dla Kandydatów

Matura i Studia

Program obejmuje przedmioty ogólnokształcące przygotowujące do matury (poziom podstawowy i rozszerzony). Absolwenci najczęściej wybierają kierunki: Informatyka, Data Science, Cyberbezpieczeństwo.

Wymagania Sprzętowe

Szkoła zapewnia pełne zaplecze sprzętowe niezbędne do realizacji podstawy programowej. Posiadanie własnego komputera jest zalecane do pracy własnej w domu.
